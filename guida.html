1. Accesso

Apri il browser e vai all'indirizzo http://localhost:5000/. Inserisci le tue credenziali per accedere al pannello amministrativo.

2. Navigazione

Usa la sidebar per navigare tra le diverse sezioni:

  • Dashboard: riepilogo statistiche e stato generale.
  • Importazioni: importazione e aggiornamento dati Rider e Gare.
  • Stagioni & Round: gestione stagioni e round delle competizioni.
  • Team: gestione dei team registrati.
  • AI Tools: suggerimenti di formazione AI (Beta).

3. Importazione e aggiornamento dati Rider

L'importazione dei dati ora avviene tramite file CSV o JSON.

3.1 Formati supportati

  • CSV: intestazioni obbligatorie name, team, email
  • JSON: array di oggetti con chiavi coerenti, es. [{"name":"Mario Rossi","team":"Team A","email":"mario@example.com"}]

3.2 Procedura

  1. Apri Importa Rider dalla sidebar.
  2. Clicca su Scegli file e seleziona il file CSV o JSON.
  3. Clicca su Carica per visualizzare l'anteprima dei dati.
  4. Controlla i dati nella tabella di anteprima.
  5. Clicca su Conferma Importazione per salvare o aggiornare i rider nel database.

3.3 Aggiornamento automatico

4. Importazione e gestione Gare / Round

Anche le gare e i round possono essere importati tramite file CSV/JSON.

  • File CSV con colonne come round_number, season, name, start_date, end_date
  • Anteprima dei round prima di salvare
  • Conferma finale per aggiornare il database

5. Messaggi e notifiche

Tutte le azioni mostreranno messaggi flash in alto:

  • success ‚Üí operazione completata
  • warning ‚Üí dati mancanti o non validi
  • danger ‚Üí errore durante l'importazione

6. Suggerimenti pratici

  • Verifica che i file CSV/JSON siano ben formattati prima di caricarli.
  • Usa l'anteprima per controllare eventuali errori o incongruenze.
  • Non modificare manualmente il database per evitare conflitti.
